# Level 3: Workshop

파일과 폴더를 직접 다루는 법을 배우는 퍼즐입니다.
17 개의 방 중 올바른 순서로 이동하며 `mkdir`, `cp`, `mv`, `rm` 을 배웁니다.

## 시작하기

아래 명령어를 순서대로 입력하세요:

```
cd start
cat note
```

각 방의 안내를 따라 과제를 풀고, 정답에 해당하는 room 으로 이동하세요.
틀린 답을 고르면 막다른 길에 도착합니다. 힌트를 읽고 돌아와서 다시 시도하세요.

실수해서 꼬였다면 언제든 처음부터 다시 시작할 수 있습니다:

```
cd ~
git checkout world-level3
cd start
cat note
```
